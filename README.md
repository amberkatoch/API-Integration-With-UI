# API Integration with UI

## Objective

Build a user interface that fetches data from a REST API and displays it dynamically.

## Features

- REST API Integration
- Async/Await Data Fetching
- Dynamic UI Rendering
- Error Handling
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API

## API Used

https://jsonplaceholder.typicode.com/users
